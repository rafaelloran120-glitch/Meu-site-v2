Curiosidades sobre o java☕️

  • O Java foi criado em 1995 por James Gosling na Sun Microsystems.
  • O nome “Java” veio de um café da Indonésia, porque os criadores eram viciados em café.
  • O slogan original do Java era “Write once, run anywhere” (escreva uma vez, rode em qualquer lugar).
  • Minecraft, um dos jogos mais famosos do mundo, foi programado em Java.
  • Mais de 3 bilhões de dispositivos no mundo rodam Java (incluindo celulares antigos, caixas eletrônicos e até TVs).

🐍Curiosidades sobre Python.

  • O Python foi criado em 1991 por Guido van Rossum.
  • O nome “Python” veio do grupo de comédia britânico Monty Python, não da cobra.
  • Python é a linguagem mais usada em inteligência artificial e ciência de dados hoje.
  • A sintaxe do Python foi feita pra ser simples e parecida com a linguagem humana.
  • Empresas como Google, Netflix, Instagram e Spotify usam Python em seus sistemas.
